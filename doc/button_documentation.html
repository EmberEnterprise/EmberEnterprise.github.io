    Button - Documentation

    Easy to use and customizable button component.

    API

    Attribute Type Value Description
    autofocus boolean false default: Average button
    true Button automatically gets focused when page loads
    disabled boolean false default: Can click button
    true Button is unusable and un-clickable
    on-click string (action <action-name>) Triggers associated action when the button is clicked
    text string <button-text> Text do display on the button. Not necessary when used in block-form
    size string extra-small
    small
    medium default
    large
    aspect string default default: Average button
    primary Provides extra visual weight and identifies the primary action in a set of buttons
    success Indicates a successful or positive action
    info Contextual button for informational alert messages
    warning Indicates caution should be taken with this action
    danger Indicates a dangerous or potentially negative action

    Examples

    Block-form

    {{#e-button}}
    My Button
{{/e-button}}

    Inline

    {{e-button text="By Button"}}
